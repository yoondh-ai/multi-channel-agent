## 🚀 Groq API 키 발급 (30초 소요)

### 1단계: 회원가입
1. https://console.groq.com 접속
2. "Sign Up" 클릭
3. 이메일 또는 Google 계정으로 가입
4. 이메일 인증 완료

### 2단계: API 키 생성
1. 로그인 후 왼쪽 메뉴에서 "API Keys" 클릭
2. "Create API Key" 버튼 클릭
3. 키 이름 입력 (예: "marketing-agent")
4. "Submit" 클릭
5. 생성된 API 키 복사 (한 번만 표시됨!)

**API 키 형식:** `gsk_...` (약 56자)

### 3단계: Streamlit Cloud에 설정

#### Streamlit Cloud에서:
1. 앱 대시보드 접속
2. 앱 선택 → "Settings" (⚙️) 클릭
3. "Secrets" 탭 선택
4. 다음 내용 추가:

```toml
GROQ_API_KEY = "gsk_여기에_실제_키_입력"
```

5. "Save" 클릭
6. 앱 자동 재시작 대기 (약 30초)

#### 로컬 개발 환경에서:
`.env` 파일에 추가:
```bash
GROQ_API_KEY=gsk_여기에_실제_키_입력
```

## 🔧 고급 설정 (선택사항)

### 웹 검색 API 추가

기본적으로 DuckDuckGo를 사용하지만, 더 나은 검색 결과를 원하면:

#### Serper API (Google 검색)
```toml
SERPER_API_KEY = "your-serper-key"
```
- https://serper.dev (무료 티어: 2,500 검색/월)

#### Tavily API
```toml
TAVILY_API_KEY = "your-tavily-key"
```
- https://tavily.com (무료 티어: 1,000 검색/월)

## 🐛 문제 해결

### "Groq 초기화 실패" 오류
1. API 키 형식 확인 (`gsk_`로 시작)
2. Secrets에 공백 없이 입력했는지 확인
3. 앱 재시작

### "웹 검색 실패" 경고
- 정상입니다! 검색 실패해도 AI가 콘텐츠 생성
- 더 나은 결과를 원하면 Serper/Tavily API 추가

### 콘텐츠가 여전히 샘플 데이터
1. 앱 상단 메시지 확인
2. "✅ Groq AI 연결됨" 표시되는지 확인
3. 브라우저 캐시 삭제 후 재시도
