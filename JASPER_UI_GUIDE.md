# 🎨 Jasper 스타일 UI 가이드

## ✨ 새로운 디자인

Jasper AI를 참고한 전문적이고 직관적인 UI로 완전히 개편되었습니다!

## 📐 레이아웃 구조

### 1. 사이드바 (왼쪽)
**보라색 그라데이션 배경**

#### 📋 마케팅 프레임워크
- **PAS** (Problem-Agitate-Solution)
  - 문제 제기 → 고통 증폭 → 해결책 제시
  - 제품 소개, 랜딩 페이지에 최적
  
- **AIDA** (Attention-Interest-Desire-Action)
  - 주목 → 관심 → 욕구 → 행동
  - 광고, 이메일 마케팅에 최적
  
- **BAB** (Before-After-Bridge)
  - 현재 상황 → 이상적 상황 → 연결 방법
  - 사례 연구, 변화 스토리에 최적
  
- **FAB** (Features-Advantages-Benefits)
  - 기능 → 장점 → 혜택
  - 제품 설명, 기술 문서에 최적
  
- **4P** (Picture-Promise-Prove-Push)
  - 그림 → 약속 → 증명 → 촉구
  - 세일즈 레터, 프레젠테이션에 최적

#### 🎯 타겟 독자
- C-레벨 임원
- IT 담당자
- 마케팅 담당자
- 중소기업 대표
- 일반 소비자
- 개발자
- 학생/교육자

#### 🎨 말투 (Tone)
- 전문적
- 친근한
- 유머러스
- 긴급한
- 교육적
- 감성적
- 혁신적

#### ⚙️ 생성 옵션
- 생성 개수: 1-3개
- SEO 최적화
- CTA 포함

### 2. 메인 영역 (중앙)

#### 좌측 컬럼 (50%)
**입력 폼 카드**

- **제품/서비스명**
  - 마케팅할 제품 이름
  - 예: "MarkAny 제로트러스트 솔루션"

- **핵심 특징 (3-5개)**
  - 제품의 주요 특징을 나열
  - 불릿 포인트 형식 권장
  ```
  • 완벽한 보안 아키텍처
  • 클라우드 네이티브 설계
  • 실시간 위협 탐지
  • 간편한 도입 및 관리
  • 24/7 전문가 지원
  ```

- **타겟 키워드**
  - SEO를 위한 핵심 키워드
  - 예: "제로트러스트, 클라우드 보안, 네트워크 보안"

- **추가 정보 (선택사항)**
  - 프로모션 정보
  - 경쟁 우위
  - 특별 강조 사항

#### 우측 컬럼 (50%)
**결과 및 편집기 카드**

##### 초기 상태
- 안내 메시지
- 사용 팁 표시

##### 생성 후
- **채널별 탭**
  - 📝 네이버 블로그
  - 💼 LinkedIn
  - 🐦 Twitter
  - 📧 이메일

- **버전별 서브탭**
  - 버전 1, 버전 2, 버전 3

- **메트릭 표시**
  - 글자 수
  - SEO 점수
  - 가독성

- **편집기**
  - 제목 편집
  - 본문 편집
  - 실시간 수정 가능

- **액션 버튼**
  - 📋 복사
  - 💾 다운로드
  - 🔄 재생성

## 🎨 디자인 특징

### 색상 팔레트
- **Primary**: #6366f1 (인디고)
- **Secondary**: #8b5cf6 (보라)
- **Background**: #f8f9fa (연한 회색)
- **Card**: #ffffff (흰색)
- **Text**: #1f2937 (진한 회색)
- **Muted**: #6b7280 (중간 회색)

### 그라데이션
```css
background: linear-gradient(135deg, #6366f1 0%, #8b5cf6 100%);
```

### 카드 스타일
- 흰색 배경
- 둥근 모서리 (12px)
- 부드러운 그림자
- 충분한 패딩 (2rem)

### 버튼 스타일
- 그라데이션 배경
- 호버 시 위로 이동 효과
- 그림자 강조
- 전체 너비

### 탭 스타일
- 연한 회색 배경
- 선택된 탭: 그라데이션
- 둥근 모서리
- 부드러운 전환

## 🚀 사용 흐름

### 1단계: 설정 선택 (사이드바)
```
프레임워크: PAS 선택
타겟 독자: IT 담당자 선택
말투: 전문적 선택
생성 개수: 2개
```

### 2단계: 정보 입력 (좌측)
```
제품명: MarkAny 제로트러스트
핵심 특징: 
• 완벽한 보안
• 클라우드 네이티브
• 실시간 탐지
키워드: 제로트러스트, 보안
```

### 3단계: 생성 (버튼 클릭)
```
🚀 콘텐츠 생성하기
→ AI가 4개 채널 × 2개 버전 = 8개 콘텐츠 생성
```

### 4단계: 확인 및 편집 (우측)
```
채널 선택: 네이버 블로그
버전 선택: 버전 1
→ 제목/본문 편집
→ 복사 또는 다운로드
```

## 💡 사용 팁

### 최상의 결과를 위해

1. **구체적인 제품명**
   - ❌ "보안 제품"
   - ✅ "MarkAny 제로트러스트 네트워크 보안 솔루션"

2. **명확한 특징**
   - 3-5개의 핵심 특징
   - 각 특징은 간결하게 (1줄)
   - 불릿 포인트 사용

3. **적절한 프레임워크**
   - 제품 소개 → PAS
   - 광고 캠페인 → AIDA
   - 사례 연구 → BAB
   - 기술 문서 → FAB

4. **타겟 맞춤**
   - 독자에 맞는 톤 선택
   - C-레벨 → 전문적
   - 일반 소비자 → 친근한

## 🎯 채널별 최적화

### 네이버 블로그
- 긴 형식 (1,500-2,500자)
- SEO 최적화
- 구조화된 섹션
- 이미지 삽입 위치 제안

- 전문적 톤
- 800-1,200자
- 인사이트 중심

### Twitter
- 3-7개 트윗

- 이모지 활용

### 이메일

- 500-800자
- 명확한 CTA
