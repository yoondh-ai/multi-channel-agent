**필요한 것: Groq API 키만 설정하면 됩니다!**

## 🎯 30초 설정

### 1단계: Groq API 키 발급
```
1. https://console.groq.com 접속
2. 이메일로 가입 (무료)
3. "API Keys" 클릭
4. "Create API Key" 클릭
5. 키 복사 (gsk_로 시작)
```

### 2단계: Streamlit Cloud 설정
```
1. Streamlit Cloud 앱 대시보드
2. 앱 선택 → Settings (⚙️)
3. Secrets 탭
4. 다음 추가:

GROQ_API_KEY = "gsk_여기에_복사한_키"

5. Save
6. 30초 대기 (자동 재시작)
```

### 3단계: 확인
```
앱 새로고침 → 상단에 표시:
✅ Groq AI 연결됨 - 무료로 실제 콘텐츠 생성 가능!
```

## ❓ 문제 해결

### "데모 모드" 메시지 표시
→ API 키가 설정되지 않음
→ Secrets 확인 및 앱 재시작

### "Groq 초기화 실패"
→ API 키 형식 확인 (gsk_로 시작)
→ 공백 없이 입력했는지 확인

### 콘텐츠가 여전히 샘플
→ 브라우저 캐시 삭제
→ 앱 완전히 재시작
