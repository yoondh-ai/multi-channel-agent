## 배포 방법

### GitHub Desktop
1. 변경사항 확인
2. 커밋 메시지: "Upgrade to V2: Jasper AI style platform"
3. Push origin

### Streamlit Cloud
- 자동으로 재배포됨 (2-3분 소요)

## 롤백 방법
V1으로 돌아가려면:
1. `app.py` 삭제
2. `app_v1_backup.py`를 `app.py`로 이름 변경
3. GitHub에 푸시

## 해커톤 발표 포인트

1. **문제 정의**
   - 보안 기술 문서 → 마케팅 콘텐츠 변환의 어려움
   - 채널별 맞춤 콘텐츠 제작 시간 소요

2. **솔루션**
   - AI 기반 자동 콘텐츠 생성
   - 12가지 템플릿으로 모든 채널 커버
   - 검수 및 편집 기능으로 품질 보장

3. **차별점**
   - Jasper AI 수준의 UX
   - B2B 보안 마케팅 특화
   - 브랜드 보이스 일관성 유지

4. **비즈니스 가치**
   - 콘텐츠 제작 시간 80% 단축
   - 일관된 브랜드 메시지
   - 멀티채널 동시 운영 가능
